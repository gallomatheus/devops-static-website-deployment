# Projeto de deploy de aplicação HTML estática utilizando Docker, Nginx e AWS EC2 com armazenamento da imagem no Amazon ECR.

## Tecnologias utilizadas
- Docker
- Nginx
- AWS EC2
- AWS ECR
- Linux Ubuntu
- Git

DIAGRAMA:
Developer
   ↓
Docker Build
   ↓
Amazon ECR
   ↓
Amazon EC2
   ↓
Nginx Container
   ↓
Website Online
